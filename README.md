#Scrabble

##Author

Created by Erik Tolentino

##Description

This app allows a user to enter a word, and will return that words scrabble value.

#Setup

## Installation

* git clone this repository
* Change into the new directory

## Running / Development

* Run `composer install` in the project's home directory
* Start MAMP
* In your web browser, go to `localhost:8889` to view the app
